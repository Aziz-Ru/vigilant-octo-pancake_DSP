# Static system

Digital Signal Processing (DSP), a static system (also called a memoryless system) is a system where the output at any given time depends only on the input at that exact same time and not on past or future inputs.

Let y=f(x)=x\*x

Here, the output

y is calculated directly from the input
x using the function f(x). No matter what happens before, if you input a specific
x, the output will always be x\*x
