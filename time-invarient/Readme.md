# Time-invariant system

A time-invariant system is a system where the output does not change if the input is shifted in time._For a time-invariant system, the output and input should be delayed by some time unit. Any delay provided in the input must be reflected in the output for a time invariant system._

A time-invariant system means that if we delay the input by some time t0, the output must also be delayed by the same amount, without any change in shape or amplitude.
